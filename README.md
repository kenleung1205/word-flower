# 🌸 字族開花 (Word Family Flower)

將實體「字族開花圖」變成互動小遊戲：拖曳部首積木去花心，拼出新字，花瓣逐塊開！

## 兩種模式

- **🌸 字族開花**（聲旁為主）— 花心放聲旁（也、青⋯），拖部首砌字（女＋也＝她），學形聲字一家人。
- **🧩 部首開花**（部首為主）— 花心放部首（氵、扌、木⋯），拖其他字（聲旁）砌字（氵＋青＝清、氵＋工＝江），體會「**部首表意**」：同一個部首嘅字意思相關。部首花會自動由字族資料反推，再加手寫嘅豐富部首花。

## 玩法

1. **拖曳拼字** — 將下方色彩積木拖去花心，一掂到就「嘭」一聲合體變新字，飛上花瓣
2. **即時回饋** — 「叮！」音效 + 粵語讀字 + emoji；全部開滿就放煙花 🎆。讀音係即時合成（唔係預錄），所以自己加嘅字一樣有讀音
3. **🔊 㩒花心聽讀音** — 遊戲入面㩒花心，可聽返個花心（部首模式讀部首，字族模式讀花心字）
4. **📖 圖鑑** — 收集種出嚟嘅花（字族 + 部首），㩒一㩒學過嘅字睇例詞例句
5. **📚 例詞例句** — 砌到一隻字後㩒入個字，就見到例詞同例句；內容由老師自己加／改，存喺裝置（㩒詞句仲可以粵語讀出）
6. **🎯 測驗** — 兩種：**聽詞揀字**（讀詞語揀啱字）同**部首配意思**（配部首同佢嘅意思）；自動記低分數同錯題
7. **📊 老師報告** — 圖鑑頁可睇：已開花數、學過字數、測驗正確率、最常錯嘅字／部首
8. **➕ 加字** — 兩個模式都可以自己加花：字族模式加（花心字 + 部首 + 新字），部首模式加（部首做花心 + 聲旁 + 新字），即刻試玩
9. **➕ 加字落現有花** — 每朵花左上角㩒「＋」，為同一朵花加多啲字，數目不限；加完佢會變返花蕾等你重新開花（字族模式新加嘅字會自動同時喺部首模式出現）
10. **🔄 重設** — 圖鑑頁可重設開花進度，或單獨清除測驗紀錄（自訂花唔會刪除）

## 技術

- 純 HTML / CSS / JavaScript，無 build step、無依賴
- Pointer Events 拖曳（觸控 + 滑鼠通用）
- Web Audio API 合成音效（毋須音訊檔）
- Web Speech API 粵語讀字（zh-HK）
- `localStorage` 儲存圖鑑進度
- Canvas 煙花動畫

## 本地運行

```bash
npx serve .

```
