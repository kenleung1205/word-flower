# 🌸 字族開花 (Word Family Flower)

將實體「字族開花圖」變成互動小遊戲：拖曳部首積木去花心，拼出新字，花瓣逐塊開！

## 玩法

1. **拖曳拼字** — 將下方色彩鮮豔嘅部首積木（氵、亻、女、土⋯）拖去花心嘅字（例如「也」）
2. **魔法效果** — 一掂到就「嘭」一聲合體變新字（女 + 也 = 她），然後飛上花瓣
3. **即時回饋** — 「叮！」音效 + 粵語讀字（「她！女字旁嘅她，她們嘅她！」）+ emoji 小動畫
4. **開晒花** — 全部花瓣開滿就放煙花慶祝 🎆
5. **魔法圖鑑** — 收集種出嚟嘅字族花，進度自動儲存喺裝置上
6. **➕ 加字** — 自己加新嘅字族花（花心字 + 部首/新字/詞語/emoji），即刻試玩
7. **🔄 重設** — 圖鑑頁可以重設開花進度，重新再玩（自訂字族唔會刪除）

## 字族內容

| 花心 | 花瓣 |
|------|------|
| 也 | 她 他 地 池 馳 |
| 青 | 清 晴 請 蜻 情 |
| 包 | 跑 抱 飽 泡 |
| 馬 | 媽 嗎 碼 螞 |
| 巴 | 爸 把 吧 爬 |

## 技術

- 純 HTML / CSS / JavaScript，無 build step、無依賴
- Pointer Events 拖曳（觸控 + 滑鼠通用）
- Web Audio API 合成音效（毋須音訊檔）
- Web Speech API 粵語讀字（zh-HK）
- `localStorage` 儲存圖鑑進度
- Canvas 煙花動畫

